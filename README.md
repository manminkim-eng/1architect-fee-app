## 🚀 GitHub Pages 배포 방법

저장소 루트에 아래 파일을 모두 업로드합니다.

```
📁 저장소 루트
├── index.html          ← 메인 앱 (단일 파일, React CDN)
├── manifest.json       ← PWA 매니페스트
├── sw.js               ← 서비스 워커
├── favicon.ico
├── og-image.png
└── icons/              ← PWA 아이콘 세트 (PNG 고정)
```

1. 저장소 → **Settings → Pages** → Source: Deploy from a branch → **main / root** → Save
2. 배포 URL: `https://manminkim-eng.github.io/1architect-fee-app/`
3. `sw.js` 캐시 때문에 옛 화면이 보이면 **Ctrl+Shift+R**(강력 새로고침)
